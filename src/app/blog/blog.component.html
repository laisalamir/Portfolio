4 simple steps to update your repository on GitHUB:

You can use your preferred editor terminal, Git bush installed on your device or update the repository directly on the GitHub website

1 Step: Git init

2 Step: Git add

3 Step: Git commit

Git commit -m “update name”

4 Step: Git push

git push origin main

Press F5 (Update) on your GitHub repository page and check for updates.

Como fazer Deploy do Angular no GitHub :

Neste tutorial você vai aprender como fazer deploy de projeto Angular no GitHub.

**Este é o que chamamos de Deploy manual

1 Passo: Rodar um comando de build

• Escolher um projeto Angular para fazer o submit no github

- Você pode criar um novo projeto ou escolher um repositório que já esteja disponível no seu repositório do github.

• Abra seu editor de preferência e inicie o terminal.

** No meu caso, irei utilizar o VS Code.

• Iniciar o build do seu projeto.

** Vai gerar uma pasta chamada dist.

2 passo: Gerar arquivo 404.html

• Abra o arquivo package.json

Vamos criar um script para facilitar o comando

• De volta ao terminal do projeto vamos rodar o comando : npm run deploy:ghdocs

** O terminal vai gerar uma pasta docs com o nosso index.html

• Copia e cola seu arquivo index html (dentro da pasta dist) e renomeie o nome para 404.html

3 Passo: Fazer commit

• Agora vamos fazer um commit de todas essas atualizações e um push para o repositório.

** Commit -m para enviar uma mensagem

4 Passo: Fazer um push

• Vamos “joga” o projeto para o servidor.

Git push || git push origin master

5 Passo: Settings

• Agora já no repositório do projeto no GitHub, vamos para settings

 Git hub pags

** Selecione Branch: Master  /docs  Save

** Agora vai aparecer uma mensagem a dizer que seu site esta pronto e disponível nesse endereço.
